Сереализация - это процесс перевода объекта в байтовый поток,
в результат сереализации можно получить файл с данными определённого формата

Десериализация - обратный процесс. к примеру:
Из JSON формата получаем байтовый поток и обратно конвертируем в объект

JSON.stringify(club); - в параметры передаём тот объект который нужно сереализовать

JSON формат записыват данные в формате - ключ : значение

JSON.parse(serializedClub); - Десериализация, получает строку в JSON формате, возвращает объект
