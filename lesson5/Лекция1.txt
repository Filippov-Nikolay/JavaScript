С помощью функционального подхода, можно создать
сущности в js. 
Функция фабрика - функция которая создаёт, инициализирует и 
возвращает объект. Недостаток:
 - Функции внутри объектов будут дублироватся для каждого
 экземпряла
Функция конструктор - можно рассматривать как аналаг классов в 
строго типизированных языка программирования. Функция-конструктор - 
метод не дублирует физически для каждого экземпряла

instanceof - ключ. слово, которое проверяет наличие в цепочке прототипов,
прототипов указанной функции

Рефакторинг - это контролируемый процесс улучшения кода, без
написания новой функциональности. Результат рефакторинга - это чичтый кода
и простой дизайн
