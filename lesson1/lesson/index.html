Лекция 1. Введение

Следующая леция
  • JS - язык динамеческой типизации. Тип данных меняется на основе правостороннего значения
  • Интерполяция - встраивание данных в строку. Например:
let a = 15;
console.log(`Ваш возраст - ${a}`);
  • === - Оператор тождественности проверяет и значения и типы двух переменных
let a = "10";
let b = 10;
if (a === b) {
console.log(true);
} else {
console.log(false);
}
  • ?? - Оператор позволяет проверить значени на null и underfinde. Он принимает два операнда. Возвращает значение правого операнда. Например:
let a;
let b = a ?? "Hello world";
console.log(b); // Hello world
let a = 10;
let b = a ?? "Hello world";
console.log(b); // 10
  • prompt - для ввода данных
let a = prompt("Введите данные: ");
console.log(a);
  • confirm - true/false
let a = confirm("Вы уверенны...? ");
console.log(a);
  • alert - вывод информации
alert("Hello world");
